Desde esta pantalla se podrá importar la información de un prompt de forma semi-automática.

De momento solo tenemos disponible la importación desde ChatGPT, pero estamos trabajando para incorporar otros servicios.

Para importar un prompt desde ChatGPT, siga los siguientes pasos:
  • Accede a la web de ChatGPT, pulsando aquí
  • Abre la consola de desarrollador de tu navegador (en Chrome pulsando F12)
  • Abre la sección de "Network" y ahí pulsa el filtro de "Fetch/XHR"
  • Accede al chat que quieres importar
  • Habrán aparecido algunas peticiones en la consola del desarrollador. Busca y pincha sobre la que tiene un nombre con números y letras aleatorios y tiene de path "/backend-api/conversation/..."
  • Abre la pestaña "Response" que contendrá el chat en formato Json. Copia el texto y pégamelo en el textarea de arriba
